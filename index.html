Welcome to XML-Glossary plugin!

By clicking "Add glossary" you upload the glossary file and get, in the same directory, the file main.tex. This file contains the LaTeX code for the glossary.

The "Integrate" button, instead, reads the main.tex file just created and another .tex file and, for each word in the glossary, replace all occurrences of that word with \glossaryItem{word}.
